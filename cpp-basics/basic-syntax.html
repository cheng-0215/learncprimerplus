C++基本语法 ​

WARNING

若文档有错误，请前往B站联系我

本章将介绍C++的基本语法结构，帮助你理解C++程序的基本组成部分。

程序结构 ​

C++程序主要由以下部分组成：

  • 预处理器指令（头文件）
  • 函数
  • 变量
  • 语句和表达式
  • 注释

一个简单的C++程序结构如下：

cpp
#include <bits/stdc++.h>  // 预处理器指令，万能头文件（不建议在正式项目使用万能头文件）
using namespace std;  // 命名空间声明（不建议在正式项目）
// 主函数
int main() {
    cout << "Hello, World!" << endl;  // 语句，输出一个"Hello，World!"
    return 0;  // 返回语句，返回一个0
}

预处理器指令 ​

预处理器指令以#开头，用于在编译前处理源代码。常见的预处理器指令包括：

  • #include：包含头文件
  • #define：定义宏
  • #ifdef、#ifndef、#endif：条件编译
cpp
#include <iostream>  // 包含标准输入输出流头文件
#include <string>    // 包含字符串处理头文件

#define PI 3.14159   // 定义宏常量

命名空间 ​

命名空间是C++中用来组织代码的一种方式，可以避免名称冲突。标准库中的所有组件都定义在std命名空间中。

cpp
// 方法1：使用整个命名空间（不推荐在大型项目中使用）
using namespace std;

// 方法2：使用特定的名称（推荐）
using std::cout;
using std::endl;

// 方法3：直接使用完全限定名称（最安全）
std::cout << "Hello" << std::endl;

注意

在大型项目中，使用using namespace std;可能会导致名称冲突，建议使用方法2或方法3。

主函数 ​

每个C++程序都必须有一个main函数，它是程序的入口点。

cpp
int main() {
    // 程序代码
    return 0;  // 返回值0表示程序正常结束
}

main函数也可以接受命令行参数：

cpp
int main(int argc, char* argv[]) {
    // argc: 参数数量
    // argv: 参数数组
    return 0;
}

语句和表达式 ​

  • 语句是C++程序的基本执行单位，以分号(;)结束
  • 表达式是产生值的代码片段
cpp
int x;           // 声明语句
x = 5;           // 赋值语句
x = 5 + 3;       // 5 + 3是表达式，整行是语句
cout << x;       // 输出语句
if (x > 0) {}    // 条件语句

注释 ​

C++支持两种注释方式：

cpp
// 这是单行注释

/*
这是多行注释
可以跨越多行
*/

标识符 ​

标识符是用来命名变量、函数、类等程序实体的名称。C++标识符的命名规则：

  • 由字母、数字和下划线组成
  • 必须以字母或下划线开头
  • 区分大小写
  • 不能使用C++关键字作为标识符
cpp
int age;        // 有效的标识符
int _count;      // 有效的标识符
int 1value;      // 无效：不能以数字开头
int class;       // 无效：class是关键字

空白字符 ​

C++编译器通常会忽略空白字符（空格、制表符、换行符），但它们可以提高代码的可读性：

cpp
// 以下两行代码对编译器来说是等价的
int sum=a+b;
int sum = a + b;

基本输入输出 ​

C++使用流进行输入输出操作：

  • cout：标准输出流，用于向屏幕输出数据
  • cin：标准输入流，用于从键盘读取数据
  • cerr：标准错误流，用于输出错误信息
cpp
#include <iostream>
using namespace std;

int main() {
    int age;
    string name;
    
    // 输出
    cout << "请输入你的名字: ";
    
    // 输入
    cin >> name;
    
    cout << "请输入你的年龄: ";
    cin >> age;
    
    // 连续输出
    cout << "你好，" << name << "! 你今年" << age << "岁。" << endl;
    
    // 错误输出
    cerr << "这是一条错误信息" << endl;
    
    return 0;
}
