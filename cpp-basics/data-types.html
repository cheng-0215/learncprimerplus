C++数据类型 ​

数据类型是编程语言中的基础概念，它定义了变量可以存储的数据种类以及可以对这些数据执行的操作。C++提供了丰富的内置数据类型，本章将详细介绍这些类型。

基本数据类型 ​

C++的基本数据类型可以分为以下几类：

  1. 整型
  2. 浮点型
  3. 字符型
  4. 布尔型
  5. 空类型

整型 ​

整型用于存储整数值，C++提供了多种整型，它们的区别在于所占内存空间和表示范围：

类型大小范围
short至少2字节-32,768 到 32,767
int至少2字节，通常4字节-2,147,483,648 到 2,147,483,647
long至少4字节-2,147,483,648 到 2,147,483,647
long long至少8字节-9,223,372,036,854,775,808 到 9,223,372,036,854,775,807

每种整型都有对应的无符号版本，以unsigned开头，只能表示非负数，但正数范围更大：

cpp
unsigned int a = 10;  // 只能存储0及正数

提示

整型的具体大小和范围可能因编译器和平台而异。可以使用sizeof运算符查看特定类型的大小。

cpp
#include <iostream>
using namespace std;

int main() {
    cout << "int的大小: " << sizeof(int) << "字节" << endl;
    cout << "long的大小: " << sizeof(long) << "字节" << endl;
    return 0;
}

浮点型 ​

浮点型用于表示带小数点的数值：

类型大小精度范围
float4字节7位有效数字约±3.4E±38
double8字节15位有效数字约±1.7E±308
long double至少8字节，通常12或16字节通常18位有效数字因实现而异
cpp
float pi = 3.14f;       // f后缀表示float类型
double e = 2.71828;      // 默认小数是double类型
long double ld = 3.14L;  // L后缀表示long double类型

注意

浮点数可能存在精度问题，不适合用于需要精确计算的场景（如金融计算）。

字符型 ​

字符型用于存储单个字符：

cpp
char ch = 'A';  // 单引号表示字符字面量

在C++中，char类型实际上是一个小整数，可以进行算术运算：

cpp
char ch = 'A';
cout << ch << endl;         // 输出: A
cout << (int)ch << endl;    // 输出: 65 (ASCII码)
ch = ch + 1;
cout << ch << endl;         // 输出: B

C++还提供了wchar_t、char16_t和char32_t类型，用于表示宽字符和Unicode字符。

布尔型 ​

布尔型(bool)用于表示逻辑值，只有两个可能的值：true和false：

cpp
bool isValid = true;
bool hasError = false;

在条件表达式中，非零值被视为true，零值被视为false：

cpp
int x = 10;
if (x) {  // x非零，条件为true
    cout << "x是true" << endl;
}

x = 0;
if (!x) {  // x为零，!x为true
    cout << "x是false" << endl;
}

空类型 ​

void类型表示没有值，通常用于以下场景：

  1. 函数不返回值：

    cpp
    void printMessage() {
    cout << "Hello" << endl;
    // 无需return语句
}

  2. 通用指针类型：

    cpp
    void* ptr;  // 可以指向任何类型的数据

派生数据类型 ​

派生数据类型是基于基本数据类型构建的，包括：

  1. 数组
  2. 指针
  3. 引用
  4. 函数类型

这些类型将在后续章节中详细介绍。

用户自定义类型 ​

C++允许用户定义自己的数据类型，包括：

  1. 类(class)
  2. 结构体(struct)
  3. 联合体(union)
  4. 枚举(enum)

枚举类型 ​

枚举是一种用户自定义类型，用于定义一组命名的整型常量：

cpp
// 传统枚举
enum Color { RED, GREEN, BLUE };  // RED=0, GREEN=1, BLUE=2

// 可以指定值
enum Status { ERROR = -1, OK = 0, WARNING = 1 };

// C++11引入的枚举类（更类型安全）
enum class Fruit : int { APPLE, BANANA, ORANGE };

使用枚举：

cpp
Color c = RED;
Status s = OK;
Fruit f = Fruit::APPLE;  // 枚举类需要使用作用域解析运算符

类型转换 ​

C++支持两种类型转换：隐式转换和显式转换。

隐式转换 ​

当不同类型的数据进行混合运算时，编译器会自动进行类型转换：

cpp
int i = 10;
double d = i;  // int隐式转换为double

转换规则通常遵循"较小的类型转换为较大的类型"原则，以避免数据丢失。

显式转换（类型转换） ​

C++提供了多种显式类型转换方式：

  1. C风格转换：

    cpp
    double d = 3.14;
int i = (int)d;  // d被截断为3

  2. 函数风格转换：

    cpp
    float f = float(i);  // int转换为float

  3. C++风格转换（更安全，推荐使用）：

    cpp
    // 静态转换
double d = 3.14;
int i = static_cast<int>(d);  // 用于基本类型间的转换

// 动态转换
// 用于类层次间的安全向下转换，运行时检查
// 将在面向对象编程章节详细介绍

// 常量转换
const int c = 10;
int* ptr = const_cast<int*>(&c);  // 移除常量性

// 重新解释转换
// 用于不相关类型间的转换，非常危险
int* p = new int(65);
char* ch = reinterpret_cast<char*>(p);  // 将int*解释为char*

警告

不恰当的类型转换可能导致数据丢失或程序行为异常。尤其是reinterpret_cast和const_cast应谨慎使用。

类型别名 ​

C++允许为现有类型创建别名，提高代码可读性：

cpp
// 使用typedef（传统方式）
typedef unsigned long ulong;
ulong val = 100;

// 使用using（C++11引入，更灵活）
using Integer = int;
Integer count = 10;
