函数 ​

函数是C++程序的基本构建块，它是一组执行特定任务的语句。本章将详细介绍C++中函数的定义、调用、参数传递和返回值等内容。

函数的基本概念 ​

函数是一段可重用的代码，它接受输入（参数），执行特定操作，并可能返回结果。使用函数可以：

  • 将复杂的问题分解为更小、更易管理的部分
  • 避免代码重复
  • 提高代码可读性和可维护性
  • 便于团队协作开发

函数的定义 ​

C++函数的定义语法如下：

cpp
返回类型 函数名(参数列表) {
    // 函数体
    // 执行特定任务的语句
    return 表达式;  // 可选，根据返回类型决定
}

例如，一个计算两个整数之和的函数：

cpp
int add(int a, int b) {
    int sum = a + b;
    return sum;
}

函数定义的各个部分：

  1. 返回类型：函数返回值的数据类型，如果函数不返回值，则使用void
  2. 函数名：标识符，遵循变量命名规则
  3. 参数列表：函数接受的输入，可以为空
  4. 函数体：执行特定任务的语句集合
  5. return语句：将值返回给调用者，如果返回类型为void，可以省略或使用不带表达式的return;

函数的调用 ​

调用函数的语法很简单：

cpp
函数名(参数值列表);

例如，调用上面定义的add函数：

cpp
int result = add(5, 3);  // result = 8

函数声明（原型） ​

函数声明告诉编译器函数的名称、返回类型和参数列表，但不包含函数体。函数声明通常放在头文件中或源文件的开头：

cpp
返回类型 函数名(参数列表);

例如：

cpp
int add(int a, int b);  // 函数声明

int main() {
    int result = add(5, 3);  // 调用函数
    cout << result << endl;
    return 0;
}

// 函数定义可以放在调用之后
int add(int a, int b) {
    return a + b;
}

提示

在函数声明中，参数名是可选的，但包含参数名可以提高代码可读性：

cpp
int add(int, int);  // 合法但不够清晰
int add(int a, int b);  // 更清晰

参数传递 ​

C++支持三种参数传递方式：

1. 值传递 ​

值传递是默认的参数传递方式，函数接收参数值的副本：

cpp
void increment(int x) {
    x++;  // 修改的是副本，不影响原始值
}

int main() {
    int a = 5;
    increment(a);
    cout << a << endl;  // 输出5，a的值没有改变
    return 0;
}

2. 引用传递 ​

引用传递允许函数直接操作调用者的变量：

cpp
void increment(int& x) {  // 注意&符号
    x++;  // 直接修改原始值
}

int main() {
    int a = 5;
    increment(a);
    cout << a << endl;  // 输出6，a的值被修改
    return 0;
}

3. 指针传递 ​

指针传递也允许函数修改调用者的变量，但语法不同：

cpp
void increment(int* x) {  // 接受指针
    (*x)++;  // 解引用并修改
}

int main() {
    int a = 5;
    increment(&a);  // 传递a的地址
    cout << a << endl;  // 输出6，a的值被修改
    return 0;
}

选择参数传递方式的建议

  • 对于小型对象（如基本类型），如果不需要修改原始值，使用值传递
  • 对于大型对象，如果不需要修改原始值，使用const引用传递（避免复制开销）
  • 如果需要修改原始值，使用引用传递（比指针传递更简洁、安全）
  • 如果需要表示"无效"或"可选"参数，可以考虑使用指针（可以为nullptr）

默认参数 ​

C++允许为函数参数指定默认值，如果调用时未提供该参数，将使用默认值：

cpp
void printMessage(string message = "Hello", int times = 1) {
    for (int i = 0; i < times; i++) {
        cout << message << endl;
    }
}

int main() {
    printMessage();             // 输出一次"Hello"
    printMessage("Hi");         // 输出一次"Hi"
    printMessage("Hey", 3);     // 输出三次"Hey"
    return 0;
}

注意

  • 默认参数必须从右到左指定，不能跳过参数
  • 默认参数通常在函数声明中指定，而不是在函数定义中

函数重载 ​

函数重载允许定义多个同名但参数列表不同的函数：

cpp
int add(int a, int b) {
    return a + b;
}

double add(double a, double b) {
    return a + b;
}

string add(string a, string b) {
    return a + b;  // 字符串连接
}

int main() {
    cout << add(5, 3) << endl;           // 调用int版本
    cout << add(3.5, 2.7) << endl;        // 调用double版本
    cout << add(string("Hello "), "World") << endl;  // 调用string版本
    return 0;
}

编译器根据参数类型和数量选择合适的函数版本。函数重载的规则：

  • 函数必须有不同的参数类型或数量
  • 仅返回类型不同不足以构成重载
  • 默认参数不能用于区分重载函数

内联函数 ​

内联函数是一种优化技术，编译器会尝试将函数调用替换为函数体，避免函数调用的开销：

cpp
inline int max(int a, int b) {
    return (a > b) ? a : b;
}

int main() {
    int result = max(5, 7);  // 编译器可能会将其展开为：int result = (5 > 7) ? 5 : 7;
    return 0;
}

内联函数适用于简短、频繁调用的函数，但只是对编译器的建议，编译器可能会忽略内联请求。

递归函数 ​

递归函数是调用自身的函数：

cpp
int factorial(int n) {
    if (n <= 1) {  // 基本情况（终止条件）
        return 1;
    } else {  // 递归情况
        return n * factorial(n - 1);
    }
}

int main() {
    cout << factorial(5) << endl;  // 输出120 (5*4*3*2*1)
    return 0;
}

递归函数必须有一个基本情况（终止条件），否则会导致无限递归和栈溢出。

函数指针 ​

函数指针是指向函数的指针，可以用于存储函数地址并通过指针调用函数：

cpp
int add(int a, int b) {
    return a + b;
}

int subtract(int a, int b) {
    return a - b;
}

int main() {
    // 声明函数指针
    int (*operation)(int, int);
    
    // 指向add函数
    operation = add;
    cout << operation(5, 3) << endl;  // 输出8
    
    // 指向subtract函数
    operation = subtract;
    cout << operation(5, 3) << endl;  // 输出2
    
    return 0;
}

函数指针常用于回调函数、策略模式和实现多态行为。

Lambda表达式（C++11） ​

Lambda表达式是一种定义匿名函数对象的简洁方式：

cpp
// 基本语法：[捕获列表](参数列表) -> 返回类型 { 函数体 }

// 简单的lambda表达式
auto sum = [](int a, int b) { return a + b; };
cout << sum(5, 3) << endl;  // 输出8

// 带捕获的lambda表达式
int factor = 2;
auto multiply = [factor](int x) { return x * factor; };
cout << multiply(5) << endl;  // 输出10

// 修改捕获的变量（需要使用mutable关键字）
int counter = 0;
auto increment = [counter]() mutable { return ++counter; };
cout << increment() << endl;  // 输出1
cout << increment() << endl;  // 输出2
cout << counter << endl;      // 输出0（原始变量未修改）

// 引用捕获
auto incrementRef = [&counter]() { return ++counter; };
cout << incrementRef() << endl;  // 输出1
cout << counter << endl;         // 输出1（原始变量被修改）

Lambda表达式的捕获列表：

  • []：不捕获任何变量
  • [=]：按值捕获所有变量
  • [&]：按引用捕获所有变量
  • [x, &y]：按值捕获x，按引用捕获y
  • [=, &x]：按值捕获所有变量，但按引用捕获x
  • [&, x]：按引用捕获所有变量，但按值捕获x
