C++运算符 ​

运算符是用于执行特定操作的符号，如数学计算、比较、逻辑判断等。C++提供了丰富的运算符，本章将详细介绍这些运算符的使用方法和优先级。

算术运算符 ​

算术运算符用于执行基本的数学运算：

运算符描述示例
+加法a + b
-减法a - b
*乘法a * b
/除法a / b
%取模（余数）a % b
++自增++a 或 a++
--自减--a 或 a--
cpp
#include <iostream>
using namespace std;

int main() {
    int a = 10, b = 3;
    
    cout << "a + b = " << a + b << endl;  // 13
    cout << "a - b = " << a - b << endl;  // 7
    cout << "a * b = " << a * b << endl;  // 30
    cout << "a / b = " << a / b << endl;  // 3（整数除法）
    cout << "a % b = " << a % b << endl;  // 1（余数）
    
    // 浮点数除法
    double c = 10.0, d = 3.0;
    cout << "c / d = " << c / d << endl;  // 3.33333（浮点数除法）
    
    return 0;
}

自增和自减运算符 ​

自增(++)和自减(--)运算符有前缀和后缀两种形式，它们的区别在于运算的时机：

cpp
int a = 5;
int b, c;

b = ++a;  // 先自增，再赋值：a=6, b=6
c = a++;  // 先赋值，再自增：a=7, c=6

注意

自增和自减运算符的前缀和后缀形式在表达式中的行为不同，使用时需要特别注意。

关系运算符 ​

关系运算符用于比较两个值，返回布尔值（true或false）：

运算符描述示例
==等于a == b
!=不等于a != b
>大于a > b
<小于a < b
>=大于等于a >= b
<=小于等于a <= b
cpp
int a = 10, b = 20;

cout << (a == b) << endl;  // 0（false）
cout << (a != b) << endl;  // 1（true）
cout << (a > b) << endl;   // 0（false）
cout << (a < b) << endl;   // 1（true）

逻辑运算符 ​

逻辑运算符用于组合条件表达式：

运算符描述示例
&&逻辑与（AND）a && b
||逻辑或（OR）a || b
!逻辑非（NOT）!a
cpp
bool a = true, b = false;

cout << (a && b) << endl;  // 0（false）
cout << (a || b) << endl;  // 1（true）
cout << (!a) << endl;      // 0（false）

短路求值 ​

逻辑运算符具有短路求值特性：

  • 对于&&运算符，如果第一个操作数为false，则不会计算第二个操作数
  • 对于||运算符，如果第一个操作数为true，则不会计算第二个操作数
cpp
int a = 5, b = 0;

// 短路求值示例
if (b != 0 && a / b > 2) {  // b=0时，不会计算a/b，避免除零错误
    cout << "条件成立" << endl;
}

位运算符 ​

位运算符用于对整数的二进制位进行操作：

运算符描述示例
&按位与a & b
|按位或a | b
^按位异或a ^ b
~按位取反~a
<<左移a << n
>>右移a >> n
cpp
int a = 5;  // 二进制：0101
int b = 3;  // 二进制：0011

cout << (a & b) << endl;  // 1（二进制：0001）
cout << (a | b) << endl;  // 7（二进制：0111）
cout << (a ^ b) << endl;  // 6（二进制：0110）
cout << (~a) << endl;     // -6（取反，具体值取决于整数表示方式）
cout << (a << 1) << endl; // 10（二进制：1010，左移1位相当于乘以2）
cout << (a >> 1) << endl; // 2（二进制：0010，右移1位相当于除以2）

提示

位运算在底层编程、优化和特定算法中非常有用，如图形处理、加密算法和嵌入式系统编程。

赋值运算符 ​

赋值运算符用于给变量赋值：

运算符描述示例等价于
=简单赋值a = b-
+=加法赋值a += ba = a + b
-=减法赋值a -= ba = a - b
*=乘法赋值a *= ba = a * b
/=除法赋值a /= ba = a / b
%=取模赋值a %= ba = a % b
&=按位与赋值a &= ba = a & b
|=按位或赋值a |= ba = a | b
^=按位异或赋值a ^= ba = a ^ b
<<=左移赋值a <<= na = a << n
>>=右移赋值a >>= na = a >> n
cpp
int a = 10;

a += 5;  // a = 15
a -= 3;  // a = 12
a *= 2;  // a = 24
a /= 4;  // a = 6

条件运算符（三元运算符） ​

条件运算符?:是C++中唯一的三元运算符，用于根据条件选择不同的值：

cpp
条件表达式 ? 表达式1 : 表达式2

如果条件表达式为true，则整个表达式的值为表达式1的值；否则为表达式2的值。

cpp
int a = 10, b = 20;
int max = (a > b) ? a : b;  // max = 20

// 等价于if-else语句
int max;
if (a > b) {
    max = a;
} else {
    max = b;
}

其他运算符 ​

sizeof运算符 ​

sizeof运算符返回变量或类型的字节大小：

cpp
cout << sizeof(int) << endl;      // 通常为4（取决于编译器和平台）

int arr[10];
cout << sizeof(arr) << endl;      // 40（10个int元素，每个4字节）

class MyClass { int a; double b; };
cout << sizeof(MyClass) << endl;  // 通常为16（取决于内存对齐）

逗号运算符 ​

逗号运算符,允许在一个表达式中执行多个操作，整个表达式的值是最后一个表达式的值：

cpp
int a, b;
a = (b = 3, b + 2);  // 先执行b=3，再执行b+2，最后a=5

// 常用于for循环
for (int i = 0, j = 10; i < j; i++, j--) {
    cout << i << " " << j << endl;
}

成员访问运算符 ​

  • .：用于访问对象的成员
  • ->：用于通过指针访问对象的成员
cpp
class Point {
public:
    int x, y;
};

Point p1;
p1.x = 10;  // 使用.运算符

Point* p2 = new Point();
p2->y = 20;  // 使用->运算符

作用域解析运算符 ​

::运算符用于访问命名空间、类或枚举中的成员：

cpp
namespace MyNamespace {
    int value = 10;
}

class MyClass {
public:
    static int value;
};
int MyClass::value = 20;

cout << MyNamespace::value << endl;  // 10
cout << MyClass::value << endl;      // 20

运算符优先级和结合性 ​

运算符优先级决定了复合表达式中运算符的计算顺序。下表按优先级从高到低列出了C++运算符（部分）：

优先级运算符结合性
1::（作用域解析）左到右
2++ --（后缀）、()（函数调用）、[]（数组下标）、.、->左到右
3++ --（前缀）、+ -（一元）、! ~、*（解引用）、&（取地址）、sizeof右到左
4* / %左到右
5+ -（二元）左到右
6<< >>左到右
7< <= > >=左到右
8== !=左到右
9&（按位与）左到右
10^（按位异或）左到右
11|（按位或）左到右
12&&（逻辑与）左到右
13||（逻辑或）左到右
14?:（条件）右到左
15= += -= 等（赋值）右到左
16,（逗号）左到右

提示

当不确定运算符优先级时，可以使用括号()明确指定计算顺序，这样可以提高代码可读性并避免错误。

cpp
int a = 5, b = 3, c = 2;

// 不使用括号，依赖运算符优先级
int result1 = a + b * c;  // 11（先乘后加）

// 使用括号明确指定顺序
int result2 = (a + b) * c;  // 16（先加后乘）
