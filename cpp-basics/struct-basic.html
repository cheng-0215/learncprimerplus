struct结构体 ​

WARNING

若文档有错误，请在Github上编辑此页

结构体（struct）是C++中一种用户自定义的复合数据类型，它允许我们将不同类型的数据组合在一起，形成一个新的数据类型。

结构体的定义 ​

使用struct关键字定义结构体：

cpp
struct Student {
    string name;    // 姓名
    int age;        // 年龄
    double score;   // 分数
};

结构体变量的创建和初始化 ​

创建结构体变量有多种方式：

cpp
// 方式1：定义后赋值
Student s1;
s1.name = "张三";
s1.age = 18;
s1.score = 95.5;

// 方式2：定义时初始化
Student s2 = {"李四", 19, 92.0};

// 方式3：使用统一初始化语法（C++11）
Student s3{"王五", 20, 88.5};

访问结构体成员 ​

使用点运算符（.）访问结构体成员：

cpp
Student s;
s.name = "张三";
cout << "姓名：" << s.name << endl;
cout << "年龄：" << s.age << endl;

结构体数组 ​

可以创建存储多个结构体的数组：

cpp
Student class1[3] = {
    {"张三", 18, 95.5},
    {"李四", 19, 92.0},
    {"王五", 20, 88.5}
};

// 访问数组中的结构体成员
cout << class1[0].name << endl;  // 输出：张三

结构体指针 ​

可以使用指针指向结构体：

cpp
Student s = {"张三", 18, 95.5};
Student* ptr = &s;

// 使用箭头运算符（->）通过指针访问成员
cout << ptr->name << endl;    // 等价于 (*ptr).name
cout << ptr->age << endl;     // 等价于 (*ptr).age

嵌套结构体 ​

结构体中可以包含其他结构体：

cpp
struct Address {
    string city;
    string street;
};

struct Student {
    string name;
    int age;
    Address addr;    // 嵌套结构体
};

Student s = {"张三", 18, {"北京", "朝阳路"}};
cout << s.addr.city << endl;  // 访问嵌套结构体成员

结构体作为函数参数 ​

结构体可以作为函数参数传递：

cpp
// 传值
void printStudent(Student s) {
    cout << s.name << ", " << s.age << endl;
}

// 传引用（推荐，避免复制开销）
void modifyStudent(Student& s) {
    s.age++;
}

// 传指针
void modifyStudentPtr(Student* s) {
    s->age++;
}

注意事项 ​

  1. 结构体定义后面的分号不能省略
  2. C++中的结构体可以包含成员函数
  3. 结构体的大小可能大于所有成员大小之和（因为内存对齐）
  4. 大型结构体建议使用引用或指针传递，以提高效率
