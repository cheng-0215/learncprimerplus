变量与常量 ​

变量和常量是编程中最基本的概念之一，它们用于存储和管理程序中的数据。本章将详细介绍C++中变量和常量的声明、初始化和使用方法。

变量 ​

变量是一个命名的内存位置，用于存储可以在程序执行过程中改变的数据。

变量声明 ​

C++中变量声明的基本语法是：

cpp
数据类型 变量名;

例如：

cpp
int age;        // 声明一个整型变量
double salary;   // 声明一个双精度浮点型变量
char grade;      // 声明一个字符型变量
bool isActive;   // 声明一个布尔型变量

变量初始化 ​

变量声明后可以立即初始化，也可以稍后赋值：

cpp
// 声明并初始化
int count = 0;
double pi = 3.14159;

// 先声明后赋值
int score;
score = 85;

C++提供了多种初始化变量的方式：

cpp
// 传统初始化
int a = 10;

// 构造函数初始化
int b(20);

// 统一初始化（C++11引入，推荐使用）
int c{30};
double d{2.5};

提示

统一初始化（使用花括号）可以防止窄化转换（可能导致数据丢失的转换）：

cpp
int x = 3.14;    // 允许，但会丢失小数部分
int y{3.14};     // 错误，编译器会报错

多变量声明 ​

可以在一条语句中声明多个相同类型的变量：

cpp
int x, y, z;           // 声明三个整型变量
int a = 1, b = 2, c = 3;  // 声明并初始化三个整型变量

变量命名规则 ​

变量名必须遵循以下规则：

  • 只能包含字母、数字和下划线
  • 必须以字母或下划线开头
  • 区分大小写
  • 不能使用C++关键字
  • 不能包含空格或特殊字符
cpp
int age;        // 有效
int _count;      // 有效
int 1value;      // 无效：不能以数字开头
int my-var;      // 无效：不能包含连字符
int class;       // 无效：class是关键字

变量命名约定 ​

虽然不是强制性的，但以下命名约定可以提高代码可读性：

  • 使用有意义的名称
  • 使用驼峰命名法（如studentAge）或下划线分隔（如student_age）
  • 局部变量通常以小写字母开头
  • 全局变量有时以g_前缀开头

常量 ​

常量是在程序执行过程中值不能改变的变量。C++提供了多种定义常量的方式。

使用const关键字 ​

const关键字用于声明一个不可修改的变量：

cpp
const double PI = 3.14159;
const int MAX_STUDENTS = 30;

// 尝试修改常量会导致编译错误
PI = 3.14;  // 错误：不能修改const变量

使用constexpr关键字（C++11） ​

constexpr关键字用于声明可以在编译时计算的常量：

cpp
constexpr int DAYS_IN_WEEK = 7;
constexpr int SQUARE_OF_FIVE = 5 * 5;

constexpr和const的主要区别在于，constexpr保证常量在编译时求值，而const只保证不可修改。

使用#define预处理指令 ​

虽然不推荐，但也可以使用#define预处理指令定义常量：

cpp
#define PI 3.14159
#define MAX_BUFFER_SIZE 1024

注意

使用#define定义的常量没有类型检查，可能导致难以发现的错误。现代C++推荐使用const或constexpr代替#define。

枚举常量 ​

枚举也可以用来定义一组相关的常量：

cpp
enum Color { RED, GREEN, BLUE };
enum Days { MONDAY = 1, TUESDAY, WEDNESDAY, THURSDAY, FRIDAY };

Color myColor = RED;
Days today = WEDNESDAY;  // 值为3

变量的作用域和生命周期 ​

作用域 ​

变量的作用域决定了变量在程序中的可见性：

  1. 局部作用域：在函数或代码块内部声明的变量
  2. 全局作用域：在所有函数外部声明的变量
  3. 命名空间作用域：在命名空间内声明的变量
cpp
#include <iostream>
using namespace std;

int globalVar = 100;  // 全局变量

int main() {
    int localVar = 10;  // 局部变量
    
    {
        int blockVar = 20;  // 块作用域变量
        cout << globalVar << " " << localVar << " " << blockVar << endl;
    }  // blockVar在这里超出作用域
    
    cout << globalVar << " " << localVar << endl;
    // cout << blockVar << endl;  // 错误：blockVar不在作用域内
    
    return 0;
}

生命周期 ​

变量的生命周期决定了变量在内存中存在的时间：

  1. 自动变量：函数或代码块内的局部变量，在退出作用域时自动销毁
  2. 静态变量：使用static关键字声明的变量，在程序结束前一直存在
  3. 动态变量：使用new运算符创建的变量，直到使用delete显式销毁
cpp
#include <iostream>
using namespace std;

void counter() {
    int autoVar = 0;      // 自动变量，每次调用都重新初始化
    static int staticVar = 0;  // 静态变量，保持值
    
    autoVar++;
    staticVar++;
    
    cout << "自动变量: " << autoVar << ", 静态变量: " << staticVar << endl;
}

int main() {
    counter();  // 输出：自动变量: 1, 静态变量: 1
    counter();  // 输出：自动变量: 1, 静态变量: 2
    counter();  // 输出：自动变量: 1, 静态变量: 3
    
    return 0;
}

类型推导（C++11） ​

C++11引入了auto和decltype关键字，允许编译器自动推导变量的类型。

auto关键字 ​

auto关键字让编译器根据初始化表达式推导变量类型：

cpp
auto i = 10;          // i是int类型
auto d = 3.14;        // d是double类型
auto s = "Hello";     // s是const char*类型
auto v = vector<int>{1, 2, 3};  // v是vector<int>类型

auto特别适合用于复杂类型或迭代器：

cpp
// 不使用auto
std::vector<int>::iterator it = vec.begin();

// 使用auto
auto it = vec.begin();

decltype关键字 ​

decltype关键字用于获取表达式的类型：

cpp
int x = 10;
decltype(x) y = 20;  // y的类型与x相同，即int

auto func = [](int a, int b) { return a + b; };
decltype(func(1, 2)) result = func(3, 4);  // result的类型是func返回值的类型，即int
