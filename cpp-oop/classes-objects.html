类与对象 ​

面向对象编程是C++的核心特性之一，而类和对象是面向对象编程的基础概念。本章将介绍C++中类和对象的基本概念和使用方法。

类的概念 ​

类是C++中用户自定义的数据类型，它是对象的蓝图或模板。类定义了对象的属性（数据成员）和行为（成员函数）。

类的定义 ​

cpp
class ClassName {
private:
    // 私有成员，只能在类内部访问
    int privateData;
    
protected:
    // 受保护成员，可以在派生类中访问
    int protectedData;
    
public:
    // 公有成员，可以在类外部访问
    int publicData;
    
    // 成员函数声明
    void setPrivateData(int data);
    int getPrivateData();
    void displayData();
};

成员函数的定义 ​

成员函数可以在类内部定义，也可以在类外部定义。在类外部定义时，需要使用作用域解析运算符::。

cpp
// 在类内部定义
class ClassName {
public:
    void inlineFunction() {
        // 函数体
    }
};

// 在类外部定义
void ClassName::setPrivateData(int data) {
    privateData = data;
}

int ClassName::getPrivateData() {
    return privateData;
}

void ClassName::displayData() {
    cout << "Private Data: " << privateData << endl;
    cout << "Protected Data: " << protectedData << endl;
    cout << "Public Data: " << publicData << endl;
}

对象的概念 ​

对象是类的实例，它具有类定义的属性和行为。创建对象的过程称为实例化。

创建对象 ​

cpp
// 创建对象的语法
ClassName objectName;

// 示例
class Rectangle {
private:
    int length;
    int width;
    
public:
    void setDimensions(int l, int w) {
        length = l;
        width = w;
    }
    
    int getArea() {
        return length * width;
    }
};

int main() {
    // 创建Rectangle类的对象
    Rectangle rect1;
    Rectangle rect2;
    
    // 使用对象
    rect1.setDimensions(5, 3);
    cout << "Area of rect1: " << rect1.getArea() << endl;
    
    rect2.setDimensions(8, 6);
    cout << "Area of rect2: " << rect2.getArea() << endl;
    
    return 0;
}

访问对象的成员 ​

对象的公有成员可以通过点运算符.来访问。

cpp
objectName.memberName;

如果是指向对象的指针，则使用箭头运算符->来访问成员。

cpp
Rectangle* pRect = &rect1;
pRect->setDimensions(10, 7);
cout << "Area: " << pRect->getArea() << endl;

类的构造函数 ​

构造函数是一种特殊的成员函数，在创建对象时自动调用。构造函数的名称与类名相同，没有返回类型。

cpp
class Rectangle {
private:
    int length;
    int width;
    
public:
    // 默认构造函数
    Rectangle() {
        length = 0;
        width = 0;
    }
    
    // 带参数的构造函数
    Rectangle(int l, int w) {
        length = l;
        width = w;
    }
    
    int getArea() {
        return length * width;
    }
};

int main() {
    Rectangle rect1;          // 使用默认构造函数
    Rectangle rect2(5, 3);    // 使用带参数的构造函数
    
    cout << "Area of rect1: " << rect1.getArea() << endl;
    cout << "Area of rect2: " << rect2.getArea() << endl;
    
    return 0;
}

类的析构函数 ​

析构函数是一种特殊的成员函数，在对象销毁时自动调用。析构函数的名称是类名前面加上波浪号~，没有参数和返回类型。

cpp
class Rectangle {
private:
    int* length;
    int* width;
    
public:
    // 构造函数
    Rectangle(int l, int w) {
        length = new int;
        width = new int;
        *length = l;
        *width = w;
    }
    
    // 析构函数
    ~Rectangle() {
        delete length;
        delete width;
        cout << "Rectangle destroyed" << endl;
    }
    
    int getArea() {
        return (*length) * (*width);
    }
};

析构函数主要用于释放对象占用的资源，如动态分配的内存、打开的文件等。

总结 ​

  • 类是用户自定义的数据类型，是对象的蓝图
  • 对象是类的实例，具有类定义的属性和行为
  • 类的成员可以是私有的、受保护的或公有的
  • 构造函数在创建对象时自动调用，用于初始化对象
  • 析构函数在对象销毁时自动调用，用于释放资源
