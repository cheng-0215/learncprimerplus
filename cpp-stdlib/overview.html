wo'mwom# C++标准库概述

C++标准库是C++语言的一个重要组成部分，它提供了大量的类和函数，可以帮助开发者更高效地编写程序。本章将对C++标准库进行概述，介绍其主要组成部分和功能。

什么是C++标准库 ​

C++标准库是随C++编译器一起提供的一组类和函数的集合，它们实现了许多常用的操作，如输入/输出、字符串处理、数学计算、容器、算法等。使用标准库可以大大减少开发时间，提高代码质量和可移植性。

C++标准库的主要组成部分 ​

1. 输入/输出库（IO库） ​

输入/输出库提供了对数据的读取和写入功能，包括文件IO和控制台IO。

cpp
#include <iostream>  // 标准输入输出流
#include <fstream>   // 文件流
#include <sstream>   // 字符串流

int main() {
    // 控制台输出
    std::cout << "Hello, World!" << std::endl;
    
    // 文件输出
    std::ofstream outFile("example.txt");
    outFile << "Writing to a file." << std::endl;
    outFile.close();
    
    // 字符串流
    std::stringstream ss;
    ss << "Value: " << 42;
    std::string str = ss.str();
    
    return 0;
}

2. 字符串库 ​

C++提供了std::string类，用于字符串的处理和操作。

cpp
#include <string>

int main() {
    std::string s1 = "Hello";
    std::string s2 = "World";
    
    // 字符串连接
    std::string s3 = s1 + ", " + s2 + "!";
    
    // 字符串长度
    size_t len = s3.length();
    
    // 子字符串
    std::string sub = s3.substr(0, 5);
    
    // 查找
    size_t pos = s3.find("World");
    
    return 0;
}

3. 容器库 ​

容器库提供了各种数据结构，用于存储和管理数据。

cpp
#include <vector>   // 动态数组
#include <list>     // 双向链表
#include <map>      // 关联数组（键值对）
#include <set>      // 集合
#include <queue>    // 队列
#include <stack>    // 栈

int main() {
    // 向量（动态数组）
    std::vector<int> vec = {1, 2, 3, 4, 5};
    vec.push_back(6);
    
    // 列表（双向链表）
    std::list<int> lst = {1, 2, 3};
    lst.push_front(0);
    
    // 映射（键值对）
    std::map<std::string, int> ages;
    ages["Alice"] = 30;
    ages["Bob"] = 25;
    
    // 集合
    std::set<int> numbers = {5, 2, 8, 2, 1}; // 会自动排序并去重
    
    return 0;
}

4. 算法库 ​

算法库提供了各种常用算法，如排序、搜索、转换等。

cpp
#include <algorithm>
#include <vector>

int main() {
    std::vector<int> vec = {5, 2, 8, 1, 9};
    
    // 排序
    std::sort(vec.begin(), vec.end());
    
    // 查找
    auto it = std::find(vec.begin(), vec.end(), 8);
    
    // 计数
    int count = std::count(vec.begin(), vec.end(), 2);
    
    // 最大值和最小值
    auto [min_it, max_it] = std::minmax_element(vec.begin(), vec.end());
    
    return 0;
}

5. 数值库 ​

数值库提供了数学函数和随机数生成器。

cpp
#include <cmath>     // 数学函数
#include <random>    // 随机数生成

int main() {
    // 数学函数
    double x = std::sqrt(25.0);    // 平方根
    double y = std::pow(2.0, 3.0);  // 幂运算
    double z = std::sin(3.14159);   // 正弦函数
    
    // 随机数生成
    std::random_device rd;
    std::mt19937 gen(rd());
    std::uniform_int_distribution<> dis(1, 6); // 1到6的均匀分布
    
    int random_number = dis(gen); // 生成随机数
    
    return 0;
}

6. 线程库（C++11及以后） ​

C++11引入了线程库，支持多线程编程。

cpp
#include <thread>
#include <mutex>
#include <iostream>

std::mutex mtx;

void print_message(const std::string& msg) {
    std::lock_guard<std::mutex> lock(mtx);
    std::cout << msg << std::endl;
}

int main() {
    std::thread t1(print_message, "Thread 1");
    std::thread t2(print_message, "Thread 2");
    
    t1.join();
    t2.join();
    
    return 0;
}

7. 智能指针（C++11及以后） ​

C++11引入了智能指针，用于自动管理动态分配的内存。

cpp
#include <memory>

class MyClass {
public:
    MyClass() { std::cout << "Constructor called" << std::endl; }
    ~MyClass() { std::cout << "Destructor called" << std::endl; }
    void doSomething() { std::cout << "Doing something" << std::endl; }
};

int main() {
    // 独占所有权的智能指针
    std::unique_ptr<MyClass> ptr1 = std::make_unique<MyClass>();
    ptr1->doSomething();
    
    // 共享所有权的智能指针
    std::shared_ptr<MyClass> ptr2 = std::make_shared<MyClass>();
    std::shared_ptr<MyClass> ptr3 = ptr2; // 引用计数增加
    
    // 弱引用的智能指针
    std::weak_ptr<MyClass> ptr4 = ptr2;
    
    return 0;
} // ptr1, ptr2, ptr3在这里自动释放资源

使用C++标准库的优势 ​

  1. 提高开发效率：标准库提供了许多常用功能的实现，避免了重复造轮子
  2. 代码质量：标准库的实现经过了广泛的测试和优化
  3. 可移植性：标准库在所有支持C++的平台上都可用
  4. 性能：标准库的实现通常经过了性能优化

如何使用C++标准库 ​

要使用C++标准库，只需包含相应的头文件，并使用std命名空间中的类和函数。

cpp
#include <iostream>  // 包含输入输出流头文件
#include <vector>    // 包含向量容器头文件
#include <algorithm> // 包含算法头文件

int main() {
    std::vector<int> numbers = {5, 2, 8, 1, 9};
    
    // 使用标准库算法排序
    std::sort(numbers.begin(), numbers.end());
    
    // 使用标准库输出流
    for (int num : numbers) {
        std::cout << num << " ";
    }
    std::cout << std::endl;
    
    return 0;
}

总结 ​

C++标准库是C++编程的重要组成部分，它提供了丰富的功能，可以帮助开发者更高效地编写程序。在接下来的章节中，我们将详细介绍标准库的各个组成部分，包括字符串处理、容器、算法、输入输出流和异常处理等。
